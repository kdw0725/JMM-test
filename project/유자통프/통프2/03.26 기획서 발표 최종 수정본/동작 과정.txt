1. <아두이노> 
모델링한 강의실 출입문 앞 뒤와 각 좌석에 RFID리더기를 설치한다. 
	- 학생들은 출석을 위해서 각 자리에 배치된 RFID 리더기에 태그를 한다.
	- 수업 시간이 되면 서보 모터를 통해 문을 잠근다. 
	- 출입문에 배치 된 RFID 리더기에 태그를 하여 수업 도중에 입퇴실 관리를 한다.


2. <C#, MySQL>
 RFID 리더기를 통해 읽어 들인 데이터를 C# 응용프로그램을 통해서 MySQL로 전송하여 처리한다.



3. <C#>
 C#으로 만든 응용프로그램은 관리자(교수 ID가정)로 로그인하여 강의를 등록할 수 있게 한다. 
   - 등록 된 강의를 통해서 출석 관리를 시작한다.
   - 출석 관리는 출석, 결석, 무단조퇴로 이루어지며 무단조퇴는 퇴실시 RFID 리더기에 태그한 시간을 측정하여 기준 시간동안 재입실하지 않을 시에 처리하도록 한다.



4. <JSP,TomCat // MySQL // 안드로이드 스튜디오>
 JSP, TomCat을 통해서 어플과 MySQL의 DB와 연결한다.
 - 어플은 학생이 로그인하면 자신의 출석 현황을 확인할 수 있도록 하고 관리자는 처리한 출석 현황을 볼 수 있게 한다.
   시간이 남는다면, 처리한 출석 현황을 클릭하면 해당 수업을 듣는 인원에게 처리 된 출석 현황을 문자 메세지로 보내도록 한다.



추가적으로, 학생 데이터와 관리자의 데이터는 학교 측의 개인 정보 상의 이유로 제공되지 않으므로 임의의 데이터를 만들어서 시현한다.
